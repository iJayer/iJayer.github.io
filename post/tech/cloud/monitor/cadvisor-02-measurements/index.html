cAdvisor 作为监控系统的数据采集模块，那么可以采集到那些指标参数呢？

Measurements

打开 InfluxDB Web 管理界面，选择数据库：cadvisor, 执行命令：

Query: SHOW MEASUREMENTS

即可看到 cAdvisor 采集、存储在 InfluxDB 中的 Measurements

CPU

  • cpu_usage_per_cpu
  • cpu_usage_system
  • cpu_usage_total
  • cpu_usage_user
  • load_average

FileSystem

  • fs_limit
  • fs_usage

Memory

  • memory_usage
  • memory_working_set

Newwork

  • rx_bytes
  • rx_errors
  • tx_bytes
  • tx_errors
